50 git command and their uses

git--version                                -   to check if git is properly installed on your machine
git config --global user.name               -   Set the author name.
git config --global user.email              -   Set the author email.
mkdir myproject                             -   makes a new directory
cd myproject                                -   changes the current working directory
git init                                    -   to initialize the git repository
git clone                                   -   Clone a repository into a new directory.
git add                                     -   Add a file or changes in a file to the staging area.
git status                                  -   Show the status of changes as untracked, modified, or staged.
git commit -m                               -   Commit changes to the local repository with a descriptive message.
git push                                    -   Push committed changes to a remote repository.
git pull                                    -   Fetch changes from a remote repository and merge them into the current branch.
git branch                                  -   List all branches in the local repository.
git checkout                                -   Lets you navigate between the branches created by git branch 
git merge                                   -   Merge changes from one branch into the current branch.
git log                                     -   Show the commit history.
git diff                                    -   Show the differences between the working directory and the last commit.
git remote                                  -   List remote repositories.
git remote -v                               -   Show URLs of remote repositories.
git fetch                                   -   Download objects and refs from a remote repository.
git reset                                   -   Unstage changes in a file.
git reset --hard                            -   Reset the working directory to a specific commit.
git tag                                     -   Create a lightweight tag for the current commit.
git blame                                   -   Show who last modified each line of a file.
git log --oneline                           -   Show a compact commit history.
git log -p                                  -   Show the patch introduced with each commit.
git show                                    -   Show information about a commit.
git stash                                   -   Stash changes in the working directory for later use.
git stash apply                             -   Apply the most recent stash.
git cherry-pick                             -   Apply changes from a specific commit.
git remote add                              -   Add a remote repository.
git remote rm                               -   Remove a remote repository.
git fetch --prune                           -   Prune references to remote branches that have been deleted.
git branch -d                               -   Delete a local branch.
git branch -m                               -   Rename the current branch.
git log --graph                             -   Show a graph of commits and branches.
git log --since="[date]"                    -   Show commits since a specific date.
git log --author="[name]"                   -   Show commits by a specific author.
git reflog                                  -   Show a log of all changes to refs (branches, tags).
git rebase [branch]                         -   Reapply commits on top of another base commit.
git push -u [remote] [branch]               -   Set the upstream branch.
git pull --rebase                           -   Fetch and rebase changes from a remote repository.
git commit --amend                          -   Amend the last commit with new changes.
git log --grep="[pattern]"                  -   Show commits with messages matching a pattern.
git log --stat                              -   Show statistics about modified files in each commit.
git submodule add                           -   Add a submodule to the repository.
git submodule update --init                 -   Initialize and update submodules.
git clean -n                                -   Show untracked files that will be deleted.
git clean -f                                -   Delete untracked files.
git revert                                  -   Create a new commit that undoes changes from a specific commit.
